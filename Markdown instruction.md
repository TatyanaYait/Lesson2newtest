## Списки

Чтобы добавить ненумерованный список необходимо пункты выделить (*). Например, вот так:
* элемент 1
* элемент 2
* элемент 3

Чтобы добавить нумерованный список, пункты просто необходимо пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
